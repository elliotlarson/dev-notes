# Top Notes

## Starting Top

#### show only specific processes

	$ top -p 1212 -p 2310

## Interactive Commands

#### getting help

	type "h"

#### change filter

	type "F"

	then type the letter matching the field you want to sort by

#### quitting 

	type "q"
	
#### killing a process

	type "K"
	
	then type the Id of the process
